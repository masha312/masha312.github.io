How to Create a Flexible Table Component Using Auto Layout in Figma

Dec 31, 2019 | 3 min read

Creating a flexible master component for tables has surely been a challenge for me. In the ideal world, tables should be as adaptable as possible:

  • It should be easy to add and remove columns and rows
  • It should be easy to resize columns depending on the content

Fortunately, with the introduction of Auto Layout, we can now quickly create a table component that would address the two above points.
